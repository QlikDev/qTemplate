--------------------------------------------------
qRender QlikView Object Extension
--------------------------------------------------


Content:
--------------------------------------------------
qTemplate is a QlikView object extension for using Html & CSS to render QlikView data.
It is based on the excellent rendering engine jsRender (https://github.com/BorisMoore/jsrender)

You can define a rendering template either by referencing an external template file (Url) or by pasting the rendering template into the property dialog of the extension.




Instructions:
--------------------------------------------------
- Install the extension by either 
	- double-clicking the .qar file (installation for QlikView Desktop Client)
	or
	- copying the file to the extension folder of your QlikView Server

- Define at least one dimension and one expression
- Define you template for rendering the resulting data
